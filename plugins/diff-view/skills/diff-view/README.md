# diff-view

Claude Code skill for viewing file diffs in a browser.

## Features

- Side-by-side diff view
- Syntax highlighting
- Line-by-line comparison
- Works with any file type

## Usage

This skill is invoked automatically when Claude needs to show a diff between files.

See SKILL.md for detailed usage instructions.
