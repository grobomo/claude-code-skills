# Memo Edit Skill

Edit CLAUDE.md files properly.

## Rules
1. Read file first, identify sections
2. Add to existing section - NEVER create new sections
3. One-line bullets only - no paragraphs
4. Match existing style exactly
5. Show diff before applying

## Usage
```
/memo-edit <section> <rule>
```

Example:
```
/memo-edit OPERATION "Never delete - always archive"
```
